My design choices

I created my about me page to have the following (not in any particular order):
1) Info cards, supplement to picture/video that needs them, that expand towards the center of the page with additional information about the graphic. The bulk of the code was produced by ChatGPT, although I did refine bits and pieces to produce the effect that I wanted.
2) An embedded video of Twenty One, the Roblox game that I mentioned that plays on the page without opening another tab to view the video.
3) On the soccer info card, there is a link to Song Heung-Min's Wikipedia page to learn more about him and his career.
4) Borders around the graphics on the page to create a structured, cut feel. To make sure that each picture fit in the same borders, I used photoshop to edit them all to have the same width and height. For the video, I used CSS to make the width and height the same as the pictures.
5) Graphics to go along with their corresponding paragraph of text that explains my interests.
