My design choices

I created my home page to have the following: a header with the title "Robert's Website" using a cool font called Cinzel Decorative, and a navigation menu right below the header linked to other pages. Between the header/navigation menu and the footer, I have an area for the main content of the page. The footer contains a copyright notice. I used internal CSS within <style> tags and inline style="" attributes to style the content on the home page.

The page also includes three columns with hover-triggered CSS effects and buttons on each column that link to other pages, with some CSS placed in an external style sheet. This external CSS handles the image and welcome text, floats the image to the right so the text wraps around it, makes the three columns pop up with a shadow when hovered over, and sets the colors of the header, footer, and navigation menu. Additionally, I made it so the website title slightly enlarges when hovered over, and I used ChatGPT to add a flash of yellow that slides across the title on hover.
