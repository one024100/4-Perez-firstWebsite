My design choices

I created my home page to have the following:
1) A header with the title "Robert's website" using a cool font I found called Cinzel Decorative.
2) A navigation menu right below the header that is linked to other pages.
3) I have an area for the main content of the page between the header/navigation menu and the footer.
4) A footer where I have a copyright. I embedded CSS in style tags (internal) or the style="" (inline) format within tags to style the content on the home page.
5) Three columns that have CSS that triggers when hovered over, and 6) buttons on each column that have CSS that triggers when hovered over that are linked to other pages on the website. However, some of the CSS is on a separate CSS file called a style sheet (external). This includes the image and welcome text, floating the image on the right so that the text wraps around it, the three columns below that pop up and casts a shadow to catch people's eyes as their cursor wanders along, and the colors of the header, footer, and navigation menu.
6) I also made it so when the cursor hovers over the title of the website, the text gets slightly bigger. I then used ChatGPT to make it so a flash of yellow slides across the title when hovered over.
